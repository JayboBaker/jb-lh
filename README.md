# A/B testing tool

All you need is [node](https://nodejs.org/en/) which includes npm

Its recommended to use the LTS version (v6)

```
npm i
```

To run the project on localhost:8080:
```
npm start
```

To run the tests
```
npm test
```

To watch the tests
```
npm run watch-test
```

To run the production build
```
npm run build
```
